# Sentimentanalyse mit Python — Selbstlernkurs

Ein interaktiver Python-Kurs für Studierende der Geistes- und Sozialwissenschaften. 
Sentimentanalyse von Grimms Märchen als Einstieg in die Programmierung.

## Nutzung

Dieser Kurs wird über den NFDI JupyterHub bereitgestellt. 
Den Zugangslink erhalten Sie über Moodle.
